What is the question you hope to answer?Predict which users (or information sources) one user might follow in Tencent Weibo.What data are you planning to use to answer that question?Sample snapshot of Tencent weibo users’ preferences , demographics, profile keywords, follow history. There are millions of user data to explore. What do you know about the data so far?-Items are organized in categories-there are user history of who they have followed-There is an ID for the users and the recommended items-timestamps for recommendation are also provided for session analysisWhy did you choose this topic?-Interested in social media data and the interactions of user behavior-An opportunity to analyze large dimension dataset which is very common now a daysDatasource: KDDcup 2012http://www.kddcup2012.org/c/kddcup2012-track1
